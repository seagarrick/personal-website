To keep a long story short, I'm a semi-recent tech convert. Completely self-taught and self-directed, I've been making my way through as much online material as I can. I started with an Intro to Python course as Python looked like a good starter language. Next I broadened into Intro Computer Science where I cemented my abilities in Python and learned linear and complex data structures, runtime algorithms, recursion and search and sorting algorithms. From here I delved into web dev a little more specifically. I picked up JavaScript, HTML, CSS and SQL and have worked with a metric bunch of frameworks and libraries along the way (React.js, Handlebars, Node.js, NPM, MySQL, REST protocol for server requests - just to name a few). Currently I'm working on personal projects and open to new opportunities.
